## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd worldwise-website
2. Install dependencies:
    ```bash
    npm install
3. start fake api
   ```bash
    npm i json-server 
    npm run server
4. Start the development server:

    ```bash
    npm run dev
5. Open your browser
